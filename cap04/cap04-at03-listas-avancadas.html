2. Ranking de Tecnologias

  1. React

    A biblioteca de JavaScript mais popular para construção de interfaces de usuário modernas e reativas.

  2. JavaScript

    O motor de interatividade da web, essencial para qualquer desenvolvedor que deseja criar aplicações dinâmicas.

  3. CSS3 (Flexbox/Grid)

    Fundamental para o design visual e a criação de layouts complexos que se adaptam a diferentes tamanhos de tela.

  4. HTML5

    A espinha dorsal da web, responsável pela estrutura semântica e acessibilidade de todo o conteúdo online.

  5. Git & GitHub

    Ferramentas de controle de versão indispensáveis para o trabalho colaborativo e organização de código profissional.

3. Glossário Técnico

URL
Endereço Web
Caminho padrão que identifica um recurso (como uma página ou imagem) na rede mundial de computadores.
JavaScript
Linguagem de programação de alto nível usada para criar interatividade em páginas web.
Ecossistema que inclui frameworks para desenvolvimento tanto no Front-end quanto no Back-end (Node.js).
HTML
Linguagem de marcação utilizada para estruturar o conteúdo e os elementos de uma página web.
CSS
Mecanismo para adicionar estilo (cores, fontes, espaçamento) a documentos estruturados em HTML.
API
Interface que define as regras para que diferentes softwares possam se comunicar entre si.
Conjunto de ferramentas e protocolos usados para integrar serviços de terceiros em uma aplicação.
DOM
Document Object Model
Representação estruturada do HTML de uma página como uma árvore de objetos que pode ser manipulada por scripts.
HTTP
Protocolo de transferência que permite a comunicação e a troca de dados entre um cliente (navegador) e um servidor.
DNS
Sistema que traduz nomes de domínios legíveis (como google.com) em endereços IP numéricos.
